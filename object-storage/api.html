lnkto.in ➔ API

∗ Resolve original link

Make a GET call to https://lnkto.in/{id}, where {id} is a unique link identifier.

OK ➔ 301 Moved Permanently status with original link in Location header.
Not Found ➔ 404 Not Found status with JSON-formatted error in body.

# Resolving original location in Terminal with cURL

$ curl -s -D - https://lnkto.in/123456 | grep Location
Location: https://google.com

∗ Generate short link

Make a POST call to https://lnkto.in, sending "url" property in JSON body.

OK ➔ 201 Created status with shorten link in "url" body property.
Fail ➔ 400 Bad Request status with "errorCode" and "errorMessage" properties for debugging.

# Shortening link in Terminal with cURL

$ curl -X POST \
  --header "Content-Type: application/json" \
  --data '{"url": "https://www.calvertjournal.com"}' \
  https://lnkto.in

{ "success": true, "url": "https://lnkto.in/uoqorp" }
